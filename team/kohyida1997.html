Project: Dukemon

Overview of Dukemon

Figure 1. HomePage of Dukemon when first starting up.

Dukemon is a desktop app intended as a fun study tool. It is a CLI-centric (Command Line Interface) app that expands upon the idea of Flashcards to aid learning in a fun and exciting way. The main program flow of Dukemon is as follows:

  1. User creates a WordBank.

  2. User creates Cards that have a Word and Meaning each.

  3. User populates his WordBank with such Cards.

  4. User starts the Game and tries to match Meanings with Words.

  5. User completes the Game and reviews his performance Statistics.

Summary of contributions

Major Enhancements

Figure 2. UI Display of Dukemon in Game Mode, showing the different features I implemented.

  • Added a Timer and automatic Hint feature

    • Brief Description:

      • A live text and graphical countdown Timer (region in yellow box above) that shows the User how much time is left.

      • Based on the time left, Hints (presented in a Hangman style) are also automatically generated and shown to the user (region in blue box above).

    • Justification:

      • This enhancement greatly improves the product as it achieves the intended goal of creating a game-like setting. The User is also thus able to access his own performance and capability in a fun and engaging way.

      • The Hints also aids in learning, especially for weaker students or when trying out unfamiliar words.

    • Highlights:

      • Implementing this enhancement was challenging as it required seamless integration between the GUI (Graphical User Interface) and internal logical components in real time.

      • Advanced programming design concepts such as Observer Patterns, Callbacks and Functional Programming were required to preserve the quality and structural integrity of the existing code base.

      • Understanding of the JavaFX framework was also required to ensure proper synchronization, especially for testing and verification.

    • Credits (Framework/Libraries used):

      • JavaFX 11 (GUI)

      • TestFX (Testing)

      • JUnit5 (Testing)
